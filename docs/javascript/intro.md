# JavaScript 简介

欢迎来到 JavaScript 教程！

## 什么是 JavaScript？

JavaScript 是一种轻量级的编程语言，主要用于：

- 网页交互
- 前端开发
- 后端开发（Node.js）
- 移动应用开发

## 学习路线

1. 基础语法
2. 数据类型
3. 函数
4. 对象和数组
5. 异步编程
6. DOM 操作
