# 教程介绍

让我们在**5分钟内**了解 **Docusaurus**。

## 开始使用

通过**创建新站点**开始使用。

或者立即**尝试 Docusaurus**，访问 **[docusaurus.new](https://docusaurus.new)**。

### 你需要准备

- [Node.js](https://nodejs.org/en/download/) 20.0 或更高版本：
  - 安装 Node.js 时，建议勾选所有与依赖项相关的复选框。
