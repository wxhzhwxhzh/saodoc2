MySQL 零基础入门教程 —— 第一阶段：环境搭建与基本概念

面向零基础小白，用最简单的方式带你把 MySQL 环境跑起来，并理解数据库到底是什么。

📌 1. 什么是数据库？​

如果把程序比作一家小公司，那数据库就是它的“文件柜”。

  • 你要存客户信息？放数据库。
  • 要查找一个订单？查数据库。
  • 要修改密码？更新数据库。

数据库最重要的事情只有一件：帮你把数据安全、有序地保存，并随时取出来。

在众多数据库中，MySQL 是最流行、最容易上手的一种，网页、APP、小程序、后台管理系统……都离不开它。

📌 2. 开始之前：你必须知道的两个关键词​

① 数据库（Database）​

存放很多“表”的大容器，就像一个文件夹。

② 数据表（Table）​

像 Excel 表格一样，由一行行、一列列组成，用来保存一类数据。 例如：

  • 用户表（保存用户名、密码、邮箱）
  • 订单表（保存订单号、金额、时间）

理解了“数据库 = 文件夹，表 = Excel 表格”，你已经入门 30%。

📌 3. 环境搭建：让 MySQL 在你的电脑上跑起来​

我们有 3 种选择：

方案 A（最推荐）：安装 MySQL + 使用图形界面工具​

  • MySQL 服务端：负责存储数据
  • 可视化工具（如 MySQL Workbench / DBeaver）：负责让你像用 Excel 一样操作 MySQL

安装步骤（简单版）​

  1. 下载 MySQL Installer（官网下载即可）
  2. 安装时选择 "Server" 和 "Workbench"
  3. 记住设置的 root 密码（超级管理员密码）
  4. 安装完成后，启动 Workbench 即可连接数据库

你会看到一个输入密码的窗口，像这样：

MySQL Workbench  →  Local Instance  →  输入 root 密码

点一下，你就进入数据库管理界面了，这意味着：你已经成功搭建数据库环境！ 🎉

方案 B：使用压缩版 MySQL（免安装）​

适合不想安装太多东西的用户。

  • 下载 zip 版 MySQL
  • 解压
  • 双击启动 mysqld.exe
  • 再用可视化工具连接

虽然更轻量，但不适合零基础新手，略微复杂。

方案 C：使用 Docker（进阶用）​

一句命令就能跑 MySQL：

docker run -d --name mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql

更适合开发者，不建议新手一开始使用。

📌 4. 连接数据库：你要做的第一件事​

当你打开 Workbench（或其他工具），会看到一条连接：

  • Host：localhost（本机）
  • Port：3306（默认端口）
  • User：root
  • Password：你安装时设置的密码

连接成功后，就可以输入指令了，比如：

SHOW DATABASES;

表示：给我看一下当前有哪些数据库。

你会看到类似：

information_schema
mysql
performance_schema
test

这就说明——你的 MySQL 已经能正常工作了！

📌 5. 创建你的第一个数据库​

只需要一行代码：

CREATE DATABASE my_first_db;

再查看：

SHOW DATABASES;

你会看到：

my_first_db

恭喜！你已经拥有自己的第一个数据库。

📌 6. 小结（你现在已经掌握的内容）​

到这里你已经学会：

  • 数据库是什么（简单说就是一个存数据的柜子）
  • 表是什么（像 Excel）
  • 如何安装 MySQL
  • 如何连接数据库
  • 如何创建你的第一个数据库

下一阶段，我们将真正开始“动手”对数据进行增删改查，也就是 最核心的数据库技能 CRUD。
